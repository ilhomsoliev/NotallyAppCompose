
# Notally App in Jetpack Compose

Note taking Android app built with Jetpack Compose.

Technologies that are used:
- Jetpack Compose
- Dagger/Hilt
- Coroutines
- Room
- Clean Architecture
- MVVM
- DataStore
- Gradle kts
- Pretty Time
